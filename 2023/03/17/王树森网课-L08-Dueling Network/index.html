王树森强化学习网课-L08-Dueling Network

Dueling Network是DQN的替代品，之前介绍的DQN的所有改进方法都可以用在Dueling Network上。Dueling Network和DQN的区别就在于Dueling Network的网络结构和DQN不同，性能更好。

Advantage Function

最优状态价值函数

我们知道状态价值函数$V_{\pi}(s)$和策略$\pi$有关，那么肯定就存在一个最佳策略$\pi$使得此策略下的状态价值函数值最大。这个策略下的状态价值函数就叫做最佳状态价值函数$V^*(s)$，即$V^*(s) = \mathop{max}\limits_{\pi}V_{\pi}(s)$。它的表征了在状态$s$下，所有动作的价值均值最大能有多大。

优势函数

优势函数$A^*(s,a) = Q^*(s,a) - V^*(s)$，也就是最优动作价值与最优状态价值之差，表征的是采取动作$a$的最好结果和平均的最好结果能差多少。
王树森说这个优势函数最大也就能到0，即$\mathop{max}\limits_{a} A^*(s,a) = 0$。

问题
我不理解，按理说应该有动作比均值的得分高啊，为啥最大才是0？

于是我们有：
$Q^*(s,a) = V^*(s) + A^*(s,a) = V^*(s) + A^*(s,a) - \mathop{max}\limits_{a} A^*(s,a)$
添加上$\mathop{max}\limits_{a} A^*(s,a)$看似可有可无这项，为什么？我们一会儿用到了再谈。

Dueling Network

反正我们现在知道了最优动作价值函数的另一种表达式。如果我们用神经网络拟合$V^*(s)$和$A^*(s,a)$，那$Q^*(s,a)$就也被用神经网络表示了。即：
$q(s,a;\omega) = q(s,a;\omega^v,\omega^a) = V(s;\omega^v) + A(s,a;\omega^a) = V(s;\omega^v) + A(s,a;\omega^a) - \mathop{max}\limits_{a} A(s,a;\omega^a)$
因为这一个神经网络$q(s,a;\omega)$其实包含了两个神经网络$V(s;\omega^v)$和$A(s,a;\omega^a)$，与DQN的“真一个”神经网络结构不同。所以被称为Dueling Network。对他的训练和用法，跟DQN一模一样，一模一样的，就只有搭神经网络的时候结构不一样。

问题
为啥说Dueling Network结构比DQN要好呀？

为什么非要加上$\mathop{max}\limits_{a} A^*(s,a;\omega^a)$看似可有可无这项呐？
因为就算训练出来的两个网络$V(s;\omega^v)$和$A(s,a;\omega^a)$一个偏大，一个偏小，也能保证$q(s,a;\omega)$的结果八九不离十，但其实是不对的。你要说只要$q(s,a;\omega)$就行了呗，管他其他的。那我要说，强化学习不止有求价值函数这一步，其他步骤也可能需要，比如说价值函数$V(s;\omega^v)$。如果你把他训练跑偏了，其他步骤就乱套了。

所以这个问题需要解决。我们只要加入一个$\mathop{max}\limits_{a} A^*(s,a;\omega^a)$就可以理论上解决问题。因为理论上，如果$V(s;\omega^v)$和$A(s,a;\omega^a)$都训练好了，那$\mathop{max}\limits_{a} A^*(s,a;\omega^a)$就是0；如果没训练好，那$\mathop{max}\limits_{a} A^*(s,a;\omega^a)$的存在也可以保证训练不会停滞下去。
在实践上，采用$mean( A^*(s,a;\omega^a))$效果更好，即：
$q(s,a;\omega) = V(s;\omega^v) + A(s,a;\omega^a) - \mathop{max}\limits_{a} A(s,a;\omega^a)$
或
$q(s,a;\omega) = V(s;\omega^v) + A(s,a;\omega^a) - mean( A(s,a;\omega^a))$

问题
用mean更好，我就更不明白为啥了。
