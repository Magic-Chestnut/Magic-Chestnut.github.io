王树森强化学习网课-L10-Reinforcement With Baseline

第三课我们学习策略学习的时候发现了一个问题：$\frac{\partial V_{\pi}(s_t;\theta)}{\partial \theta}|{\theta = \theta_t} = Q{\pi}(s_t,a) \frac{\partial log(\pi(a|s_t;\theta))}{\partial \theta}$中的$Q_{\pi}(s_t,a)$我们并不知道。当时提出了两种方法，一种是用神经网络逼近，这种方法就是Actor-Critic Method，还有一种就是用$u_t$作为他的蒙特卡洛近似，缺点就是不能每一轮都进行更新，必须等到每一次任务结束才知道$u_t$，才能去更新参数，我们叫他REINFORCE方法。
此外，我们在第九课中学习了baseline这个小技巧，他可以在不影响蒙特卡洛近似的期望的同时，减小方差。我们现在就将baseline技巧，结合Reinforcement方法进行策略学习：
首先选择合适的baseline
王树森选择的baseline是$V_{\pi}(s_t)$，我觉得其他的baseline应该也是没问题的。现在的问题是baseline$V_{\pi}(s_t)$也是未知的。他又整了一个神经网络来近似，即$V(s_t;\omega)$。

问题：
为啥又要弄一个神经网络，直接用$V_{\pi}(s_t;\theta)$不香吗？虽然$V_{\pi}(s_t;\theta)$未知，但是也可以蒙特卡洛吧。变成$u_t \pi(a_t|s_t;\theta)$

然后我得保证对baseline的逼近逼近于baseline
那就得找个靶子，真实的basline我们也不知道，所以又要用蒙特卡洛给baseline找靶子，也就是$u_t$，因为$U_t$对$a_t$的期望就是$V_{\pi}(s_t)$。据此，我们定义逼近误差：$\delta_t = V(s_t;\omega_t) - u_t$，我们希望通过更新$\omega$参数使得$\delta$更小，所以更新率为：$\omega_{t+1} = \omega_t - \alpha \delta_t \frac{\partial V(s_t;\omega)}{\partial \omega}|{\omega = \omega_t}$。
接下来考虑更新$\theta$参数，使得$V{\pi}(s_t;\theta)$更大。也就是$\theta_{t+1} = \theta_t + \beta \frac{\partial log(\pi(a|s_t;\theta))}{\partial \theta}|{\theta = \theta_t} (u_t - V(s_t;\omega)) = \theta_t - \beta \delta_t \frac{\partial log(\pi(a|s_t;\theta))}{\partial \theta}|{\theta = \theta_t}$
