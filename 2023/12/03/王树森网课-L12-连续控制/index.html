王树森强化学习网课-L12-连续控制

之前我们所讨论的强化学习控制，不论是动作学习还是策略学习都是离散情况下的。比如说动作学习的输出是每个动作的分数，如果连续，肯定无法做到无穷维的输出；如果是策略学习，输出的是每个动作的概率值，再进行随机抽样，其实是离散的概率分布。所以我们需要单独讨论如何解决动作空间连续的问题。

方案1-连续动作空间离散化

这是最显而易见的想法，但问题很大。首先是离散化存在精度降低的问题，其次如果总的动作空间是多个子动作空间的组合（多自由度机械臂），那输出的维度会非常非常大，以至于无法训练。

方案2-确定策略梯度算法DPG

DPG的策略函数$\pi(s;\theta)$和传统的策略函数$\pi(a|s;\theta)$不同，他的输出不再是概率分布，而直接就是动作本身。

方案3-参数化策略函数

因为神经网络输出还得是有限维度，或者说不太大的维度的东西，所以无法对连续空间每一个动作都给个概率，但我们可以考虑这些工作的概率满足某些经典的分布，比如正态分布，这样整个PDF就可以用均值$\mu$和方差$\sigma$描述了。
也就是我们需要神经网络去逼近正态分布的均值：$\mu_i(s_t) = \mu_i(s_t;\theta^{\mu}t)$，用神经网络去逼近方差的对数$\rho_i = log(\sigma_i^2)$，$\rho_i(s_t) = \rho_i(s_t;\theta^{\rho}t)$。此时的策略函数为：$\pi(a_t|s_t;\theta^{\mu},\theta^{\rho})$。
其实我们用不到$\pi(a_t|s_t;\theta^{\mu},\theta^{\rho})$输出的PDF，因为只要$\mu_i(s_t)$和$\rho_i(s_t)$有了，就已经有了PDF。我们实际需要的是$\frac{\partial log(\pi(a_t|s_t,\theta))}{\partial \theta}$，其中$\theta$包含了$\theta^{\mu},\theta^{\rho}$。
根据正态分布的定义，我们知道$\pi(a|s;\theta^{\mu},\theta^{\rho}) = \prod{i=1}^d \frac{1}{\sqrt{6.28}\sigma_i} exp(-\frac{(a_i-\mu_i)^2}{2\sigma_i^2})$，进而$ln(\pi(a_t|s_t;\theta)) = \sum{i=1}^d (-ln\sigma_i - \frac{(a_i-\mu_i)^2} {2\sigma_i^2}) + const = \sum_{i=1}^d (-\frac{\rho_i}{2} - \frac{(a_i-\mu_i)^2} {2exp(\rho_i)}) + const = f(s_t,a_t;\theta) + const$，所以$\frac{\partial log(\pi(a_t|s_t,\theta))}{\partial \theta} = \frac{\partial f(s_t,a_t;\theta)}{\partial \theta}$，我们训练好$\theta$就行了，训练的方式和传统的一样。
