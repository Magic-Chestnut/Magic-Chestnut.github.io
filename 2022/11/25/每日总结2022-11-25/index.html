强化学习

强化学习确实好难啊。打算从此每天学5课内容。一周内把王树森的网课学完，然后做总结，做汇报。不管老师管不管我，都要每周做一个PDF整理一周收获和不清楚的地方，以及下周计划。

先理清楚这些概念：

  • Agent：智能体

  • enviroment：环境

  • state：agent 和 enviroment 当前的状态

  • action：智能体的抉择，比如超级玛丽可以选择上左右

  • state-transition：状态的转换

  • policy：在一定状态下，agent采取action的概率分部

  • reward：环境给智能体的奖励

  • return：打折扣的未来累计奖励

  • action value function：s下采取action后的return，Q

  • state value function：action value function对于action的均值，也就是这个s下的总未来期望，V

  • Q*：能使retrun最大的pi对应的Q

强化学习的框架下，一个智能体的运动过程这样描述：

面对环境enviroment的state，agent根据policy进行随机抽样，将抽到的action执行，enviroment根据action更新state，并且给agent一个reward。

期间，有两个随机过程：

一个是根据state选择action：这一步由 $\pi$ 函数描述，这个函数是一个s条件下的a的概率密度函数。

一个是根据action更新state：这一步是一个a条件下s的概率密度函数

强化学习要做到的是对于每一个s选择一个a，使得retrun最大

有两种方法：

  • value based learning

  • policy based learning

还有以上两种方法的结合：

  • Actor-critic method

value based learning

如果我们知道Q* 那就可以知道最好的action了。现在就是要用神经网络拟合这个Q*。用DQN的方法，求Q去接近Q*。

具体的就是TD法，其实就是变形的梯度上升法。

yt = r + gama * Qt+1 这个叫TD target

Lt就是yt和TD target的误差平方的函数，我们要让这个Lt尽量小。用梯度下降就要求梯度
